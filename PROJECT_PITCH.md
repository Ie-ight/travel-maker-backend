## 1. 기획 배경 & 핵심 주제

| 항목 | 내용 |
|------|------|
| 프로젝트를 기획하게 된 이유 | 국내 여행에 관심이 많은 팀원들이 모여, 기존 여행 플랫폼(네이버 여행, 트립어드바이저 등)이 리뷰 기반 인기 랭킹에 의존해 개인 취향을 반영한 추천이 부족하다는 한계를 직접 겪으면서 기획하게 되었습니다. "내 스타일에 맞는 여행지를 추천받을 수 있다면?"이라는 질문에서 출발해, Django + pgvector를 활용한 벡터 유사도 검색 등 실무 수준의 백엔드 기술 스택을 실제 서비스에 적용해보는 것을 목표로 삼았습니다. |
| 핵심 주제 / 서비스 방향성 | 여행 성향 테스트를 통해 사용자의 여행 스타일을 6개 축으로 분석하고, 이를 장소 데이터의 특성 벡터와 매칭해 "나와 잘 맞는 여행지"를 AI 기반으로 추천합니다. 추천에 더해 나만의 여행 경로를 만들고 다른 사람과 공유할 수 있는 커뮤니티 기능까지 제공하는 국내 여행 특화 개인화 추천 플랫폼입니다. |

## 2. 백그라운드 스토리

- **서비스가 등장하게 된 상황**
  여행을 계획할 때 블로그, SNS, 지도 앱을 오가며 정보를 수집해도, 결국 "내 취향에 맞는 곳인지"는 직접 가봐야 알 수 있다는 한계가 존재합니다. 정보는 넘치지만 나에게 맞는 정보를 가려내는 과정이 오히려 더 피로해지는 "정보 과부하" 문제에서 트래블메이커가 출발했습니다.

- **타깃 사용자 & 사용 맥락**
  국내 여행을 계획 중인 20~30대 개인 또는 소그룹 여행자로, 인기 명소보다 "내 스타일에 맞는 곳"을 찾고 싶지만 어디서부터 시작해야 할지 막막한 사용자를 주요 타깃으로 합니다. 여행 전 탐색 단계부터 경로 계획, 후기 공유까지 하나의 플랫폼에서 해결하고 싶은 사용자를 상정합니다.

- **전달하고 싶은 이야기**
  "좋은 여행지"가 아닌 "나에게 맞는 여행지"를 찾을 수 있다는 경험을 전달하고 싶습니다. 단순 인기순이 아닌 취향 기반 추천을 통해, 사용자가 자신의 여행 스타일을 발견하고 더 만족스러운 여행을 계획할 수 있도록 돕는 서비스로 기억되기를 바랍니다.

## 3. 페인포인트 & 솔루션

### 사용자가 겪는 문제
- 여행지를 검색하면 결과가 너무 많아 어디서부터 골라야 할지 모르는 정보 과부하
- 블로그·SNS·지도 앱을 따로 뒤져야 해서 정보 수집에만 많은 시간이 소요됨
- 열심히 조사해도 막상 가보면 "내 취향은 아니었다"는 경험이 반복됨

### 기존 방식의 불편함
- 네이버, 트립어드바이저 등 기존 플랫폼은 리뷰 수·평점 기반 인기 랭킹에 의존해 개인 취향 반영이 어려움
- 추천 결과가 모든 사용자에게 동일하게 노출되어 획일적인 여행 코스가 반복됨
- 장소 탐색, 경로 계획, 공유 기능이 분산되어 있어 하나의 흐름으로 이어지지 않음

### 우리 서비스의 개선 방향
- 여행 성향 퀴즈(6개 축 분석)로 사용자의 취향 벡터를 생성하고, pgvector 기반 코사인 유사도 검색으로 "나와 잘 맞는 장소"를 개인화 추천
- 퀴즈 미완료 사용자에게도 인기 장소 폴백을 제공해 진입 장벽을 낮추고, Redis 캐싱으로 응답 속도를 확보
- 장소 추천 → 경로 계획 → 공유까지 하나의 서비스 안에서 완결되는 여행 플래닝 흐름 제공

## 4. 핵심 기능 소개

### 기능 1 — 여행 취향 퀴즈

나의 여행 스타일을 진단하는 퀴즈 기능입니다.
퀴즈 응답을 **6개 축**(활동성 / 계획성 / 동행 유형 / 선호 공간 / 문화 관심도 / 예산 성향)으로 수치화하고,
그 결과로 **8가지 여행 유형** 중 하나를 도출합니다.

- 결과 화면에서 나의 유형명, 설명 텍스트, 4개의 상세 카드, 아바타 이미지를 함께 보여줍니다.
- 로그인 상태라면 결과가 자동 저장되어 이후 맞춤 추천에 반영됩니다.
- **주요 화면**: 퀴즈 진행 화면 → 결과 카드 화면

---

### 기능 2 — AI 기반 장소 추천

퀴즈 결과 벡터와 각 장소의 스타일 벡터 간 **코사인 유사도**를 계산해 취향에 맞는 장소를 추천합니다.
HNSW 인덱스(pgvector)를 활용한 ANN 검색으로 대규모 장소 데이터도 빠르게 탐색합니다.

- 태그(테마) 또는 지역으로 추가 필터링 가능합니다.
- 퀴즈 미완료 / 비로그인 사용자에게는 북마크 수 + 평점 기준 **인기 장소 폴백**을 제공하며, 결과는 Redis에 캐싱됩니다.
- **주요 화면**: 장소 목록 화면 (추천순 / 인기순)

---

### 기능 3 — 여행 경로 계획 & 공유

방문할 장소를 일자별로 묶어 나만의 여행 루트를 만들고 다른 사람과 공유하는 기능입니다.
최대 **5일, 일자당 5개 장소**를 순서대로 구성하면 지도 위에 동선이 바로 시각화됩니다.

- 지역 태그·테마 태그로 경로를 분류하고, 최신순·인기순으로 탐색할 수 있습니다.
- 리뷰 작성 시 해당 장소가 포함된 내 경로를 연결해 "어떤 루트로 다녀왔는지"를 함께 공유할 수 있습니다.
- **주요 화면**: 경로 생성 화면, 경로 상세(지도+일정) 화면, 경로 목록(탐색) 화면
