⚙ 실행 방법 (설정 변경 없이 바로 사용 가능)

  1. 프로젝트 루트에서 python -m http.server 3000 실행
  2. 브라우저에서 http://localhost:3000/console.html 접속
  3. Django 서버도 http://localhost:8000 에서 실행 중이어야 합니다

🔑 인증

카카오 소셜 로그인 · 토큰 관리 · 회원탈퇴 / 복구

POST 카카오 로그인 /api/v1/auth/kakao/login
POST 로그아웃 /api/v1/auth/logout
POST 토큰 갱신 /api/v1/auth/token/refresh

refresh_token 쿠키 자동 전송 (로그인 후 저장됨)

DELETE 회원탈퇴 /api/v1/auth/withdraw
POST 계정 복구 /api/v1/auth/recovery

탈퇴 후 14일 이내 카카오 재인증으로 복구
